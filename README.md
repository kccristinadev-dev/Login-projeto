# Login Project

This project is a simple yet effective login system implemented using **HTML**, **CSS**, and **JavaScript**. It demonstrates fundamental concepts of web development and user authentication.

## Features
- **Email Validation**: Ensures that the email entered is in a valid format.
- **Password Verification**: Checks that the password meets certain criteria to enhance security.
- **Missing Field Warnings**: Provides immediate feedback to users for any fields left empty.
- **Personalized Login Message**: Greets users with a custom message upon successful login.

## Technologies Used
- **HTML**: For the structure of the login form.
- **CSS**: For styling the form and enhancing user experience.
- **JavaScript**: To implement the login logic and validation without the use of frameworks.

## Getting Started
Follow these instructions to set up the project locally:

1. **Clone the repository**: 
   ```bash
   git clone https://github.com/kccristinadev-dev/Login-projeto.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Login-projeto
   ```

3. **Open `index.html` in your browser** to view the login form.

## Example Usage
- Enter a valid email address and password.
- Click "Login" to see the personalized message if valid.
- Test the validation by leaving fields empty or entering an invalid email format.
